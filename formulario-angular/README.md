# Angular User Registration Form

⚠️ **Projeto em desenvolvimento**

Este projeto está sendo desenvolvido para praticar **Angular**, formulários reativos e validação de dados.

## 📌 Sobre o projeto

Aplicação front-end construída com Angular que implementa um **formulário de cadastro de usuário** com validação de campos e interface baseada em Angular Material.

O objetivo é praticar conceitos importantes utilizados em aplicações profissionais.

## 🚀 Tecnologias utilizadas

* Angular
* TypeScript
* Angular Material
* Reactive Forms
* ngx-mask

## ⚙️ Como executar o projeto

Clone o repositório:

```
git clone https://github.com/wanessa-antonio/angular-forms.git
```

Entre na pasta do projeto:

```
cd angular-forms
```

Instale as dependências:

```
npm install
```

Execute o projeto:

```
ng serve
```

Abra no navegador:

```
http://localhost:4200
```
